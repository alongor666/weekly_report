# 新能源货车保险分析器

## 核心功能

专项分析新能源货车（`is_new_energy_vehicle=True`且`customer_category_3=营业货车`）的保险业务表现，提供：
- 多周趋势分析
- 机构风险评级  
- 业务类型深度钻取
- 异常波动检测

## 快速开始

```python
# 基础分析流程
analyzer = NewEnergyTruckAnalyzer()
df = analyzer.load_weekly_data(start_week=28, end_week=43)
weekly_kpis = analyzer.calculate_weekly_kpis(df)
analysis = analyzer.analyze_by_dimensions(df)
problems = analyzer.identify_problems_and_trends(weekly_kpis, analysis)
```

## 分析维度

### 1. 时间维度分析
- **周度趋势**：赔付率、保费规模、案均赔款、出险率
- **异常检测**：单周暴涨>20%、连续恶化>3周、趋势反转

### 2. 机构维度分析
- **风险评级**：按赔付率分档（低<70%、中70-80%、高>80%）
- **规模评估**：保费占比、车辆数、周均承保量
- **重点关注**：赔付率>80%且规模>100万机构

### 3. 业务类型分析
- **车型分类**：2吨以下、2-9吨、10吨以上营业货车
- **险别结构**：主全、交三、单交的赔付表现
- **续保状态**：新保、续保、转保的风险差异

## 高风险识别规则

### 立即警报条件
- 单机构赔付率>150%
- 单周赔付率环比上升>50%
- 案均赔款突增>10000元
- 单交险别赔付率>300%

### 重点关注名单
- 连续3周赔付率>100%
- 累计边际贡献率<-30%
- 规模占比>10%且赔付率>80%

## 常见问题和解决方案

### 数据质量问题
- **缺失周次**：使用插值法补充或标注缺失
- **录入错误**：赔付率>500%需人工核实
- **规模异常**：单周车辆数<20需关注数据完整性

### 分析偏差纠正
- **季节性调整**：考虑春节、国庆等长假影响
- **地区差异**：一线城市基础设施好，赔付率相对低
- **车型演进**：新能源车技术更新快，历史数据参考性降低
