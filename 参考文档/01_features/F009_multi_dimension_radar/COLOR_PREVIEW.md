## 颜色对比矩阵

不同机构颜色在雷达图上的区分度：

|          | 深蓝 | 橙色 | 绿色 | 深红 | 紫色 | 棕红 | 粉紫 |
|----------|-----|-----|-----|-----|-----|-----|-----|
| **深蓝** | ●   | ★★★ | ★★★ | ★★★ | ★★  | ★★  | ★★★ |
| **橙色** | ★★★ | ●   | ★★★ | ★★★ | ★★★ | ★★  | ★★  |
| **绿色** | ★★★ | ★★★ | ●   | ★★★ | ★★  | ★★  | ★★  |
| **深红** | ★★★ | ★★★ | ★★★ | ●   | ★★★ | ★★  | ★★  |
| **紫色** | ★★  | ★★★ | ★★  | ★★★ | ●   | ★★  | ★★  |
| **棕红** | ★★  | ★★  | ★★  | ★★  | ★★  | ●   | ★★  |
| **粉紫** | ★★★ | ★★  | ★★  | ★★  | ★★  | ★★  | ●   |

**说明**:
- ● = 自己
- ★★★ = 对比度优秀（容易区分）
- ★★ = 对比度良好（可以区分）

**结论**: 任意两种颜色都至少达到 ★★ 级别，在雷达图上能够清晰区分。

## 色盲友好测试

### 红绿色盲 (Protanopia / Deuteranopia)

在红绿色盲视角下，这7种颜色的表现：

- **深蓝 #1F77B4** → 仍为蓝色 ✅
- **橙色 #FF7F0E** → 变为黄色 ✅ (仍与其他颜色区分)
- **绿色 #2CA02C** → 变为黄褐色 ✅
- **深红 #D62728** → 变为深褐色 ✅
- **紫色 #9467BD** → 仍为紫色 ✅
- **棕红 #8C564B** → 变为深褐色 ⚠️ (与深红接近，但亮度不同)
- **粉紫 #E377C2** → 仍为粉色 ✅

**结果**: 即使在红绿色盲模式下，通过亮度、饱和度的差异，大部分颜色仍可区分。

### 蓝黄色盲 (Tritanopia)

在蓝黄色盲视角下:

- **深蓝 #1F77B4** → 变为青色 ✅
- **橙色 #FF7F0E** → 变为粉红色 ✅
- **绿色 #2CA02C** → 仍为绿色 ✅
- **深红 #D62728** → 仍为红色 ✅
- **紫色 #9467BD** → 变为粉紫色 ✅
- **棕红 #8C564B** → 仍为棕色 ✅
- **粉紫 #E377C2** → 变为淡粉色 ✅

**结果**: 蓝黄色盲视角下区分度极佳。

## 在雷达图中的实际效果

### 线条样式

```typescript
// Recharts Radar 组件配置
<Radar
  name="天府"
  dataKey="天府"
  stroke="#1F77B4"        // 描边颜色
  fill="#1F77B4"          // 填充颜色
  fillOpacity={0.08}      // 填充透明度（很低，避免重叠区域过暗）
  strokeWidth={2.5}       // 线条粗细
  dot={{
    r: 5,                 // 圆点半径
    fill: '#1F77B4',
    strokeWidth: 0,
  }}
  activeDot={{
    r: 7,                 // 悬停时圆点半径
    fill: '#1F77B4',
    stroke: '#fff',
    strokeWidth: 2,
  }}
/>
```

### 图例样式

```tsx
<Legend
  wrapperStyle={{
    paddingTop: '20px',
  }}
  iconType="line"         // 线条图标
  formatter={(value, entry) => (
    <span style={{ color: entry.color, fontWeight: 500 }}>
      {value}
    </span>
  )}
/>
```
