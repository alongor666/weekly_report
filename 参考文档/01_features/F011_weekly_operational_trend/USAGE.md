# 周度经营趋势分析 - 使用指南

## 快速开始

### 1. 访问图表
启动应用后，点击顶部导航栏的"趋势分析"标签页，即可看到周度经营趋势图表。

### 2. 基础操作

#### 查看数据
- **悬浮查看**: 鼠标悬浮在图表任意位置，Tooltip会显示该周的详细信息
- **缩放时间**: 拖动底部时间轴滑块，可缩放查看特定时间范围
- **滚轮缩放**: 在图表区域使用鼠标滚轮，可快速缩放

#### 识别风险
- **橙色大圆点**: 表示该周赔付率≥70%，为高风险周
- **淡红色背景**: 标识赔付率超过70%阈值的风险区域
- **红色虚线**: 70%赔付率警戒线
- **右上角标签**: 显示高风险周总数

#### 趋势分析
- **蓝色面积图**: 签单保费变化趋势
- **橙色点线图**: 赔付率波动情况
- **紫色虚线**: 赔付率的长期趋势方向（线性回归）

### 3. 高级功能

#### 下钻分析
点击任意橙色风险点，系统将：
1. 在控制台输出该周的详细数据
2. 弹窗提示下钻分析入口
3. （未来）自动筛选该周数据并跳转到详情页

#### 经营摘要
图表顶部自动生成的经营话语包含：
- 当前周次
- 年度累计签单保费（取最新周的累计值）
- 高风险周数统计（连续或累计周数）
- 经营状况评价

## 数据解读

### 【重要】数据含义理解

#### CSV原始数据特性
**关键认知**：每周的数据是**年度累计值**（从1月1日到该周结束的累计），而非单周值。

例如：
- 第1周数据 = 1月1日-1月4日累计
- 第42周数据 = 1月1日-10月18日累计
- 第52周数据 = 全年累计

#### 图表展示逻辑
根据顶部筛选器的"数据视图"设置，图表有两种展示模式：

**1. 当周值模式** (默认)
- 签单保费显示：累计值曲线（持续上升）
- 赔付率显示：累计赔付率（趋于稳定）
- 适用场景：查看年初至今的整体表现

**2. 周增量模式**
- 签单保费显示：每周新增量（波动曲线）
- 赔付率显示：仍是累计赔付率（不是单周！）
- 适用场景：查看每周业务增长情况

### 指标说明

#### 签单保费
- **单位**: 万元
- **含义**（取决于模式）:
  - 当周值模式：年初至今的累计保费
  - 周增量模式：本周新增保费（当周累计 - 上周累计）
- **趋势**:
  - 当周值模式：正常应持续上升
  - 周增量模式：波动是正常的，关注平均水平

#### 赔付率（关键）
- **单位**: 百分比（%）
- **计算**: **累计赔款 ÷ 累计保费 × 100%** （始终基于累计数据）
- **阈值**: 70%（行业经验值）
- **风险评估**:
  - < 60%: 优秀（盈利能力强）
  - 60-70%: 良好（正常经营）
  - 70-80%: 预警（需关注）
  - ≥ 80%: 高风险（可能亏损）

**重要提示**：
- 赔付率显示的是"从年初到该周的累计赔付率"
- 不是"该周单独的赔付率"
- 累计赔付率更稳定、更能反映整体风险水平

#### 趋势线
- **算法**: 线性回归（最小二乘法）
- **含义**: 赔付率的长期趋势方向
- **解读**:
  - 向上倾斜: 赔付率呈上升趋势，风险增加
  - 水平: 赔付率相对稳定
  - 向下倾斜: 赔付率呈下降趋势，风险控制有效

### Tooltip字段解读

#### 基础信息
- **周次**: 如"2025-W42"（2025年第42周）
- **签单保费**: 该周签订的保单保费总额（万元）
- **赔付率**: 该周的赔付率（%）

#### 风险评估
- **与阈值差值**: 赔付率与70%阈值的差值
  - 正值（+Xpp）: 超过阈值，风险
  - 负值（-Xpp）: 低于阈值，安全
  - pp: percentage point（百分点）

### 摘要中的风险原因文本
当最新周赔付率触发预警时，顶部经营摘要会自动追加“风险聚焦”段落，按业务类型 / 三级机构 / 险别组合列出 Top3 风险贡献组合，并给出赔付率、环比变化和赔款变化的信息，便于快速定位根因。

### 摘要段落结构
摘要按“经营概览 → 赔付趋势 → 业务类型异常 → 机构集中区域 → 管理建议 → 后续跟踪”六段式输出，所有结论均依据当前筛选条件（年度、机构、险别等）和最新周数据实时计算，可直接用于正式经营汇报或派发整改任务。

## 业务场景

### 场景1: 日常监控
**目标**: 快速了解本周经营状况

**操作步骤**:
1. 打开趋势分析页面
2. 查看右上角经营摘要
3. 观察最新周次（最右侧）的赔付率点位
4. 判断是否在风险区（橙色高亮）

**决策依据**:
- 无风险点 → 继续保持
- 出现风险点 → 进入场景2

### 场景2: 风险诊断
**目标**: 分析高赔付率的具体原因

**操作步骤**:
1. 定位橙色风险点
2. 阅读顶部经营摘要中的风险聚焦段落，了解Top3组合
3. 点击风险点进入下钻分析
4. 按车型/机构维度拆解数据
5. 定位高风险细分群体

**决策依据**:
- 特定车型问题 → 调整承保政策
- 特定机构问题 → 加强风控培训
- 整体性问题 → 审视定价策略

### 场景3: 趋势预判
**目标**: 预测未来经营风险

**操作步骤**:
1. 观察紫色趋势线方向
2. 对比趋势线与实际值的偏离度
3. 识别经营拐点（趋势反转的周次）
4. 分析连续高风险周数

**决策依据**:
- 趋势线向上 + 实际值持续超线 → 警惕风险累积
- 趋势线向下 + 风险周减少 → 风控措施有效
- 趋势线反转 → 关注政策/市场变化

### 场景4: 同比分析
**目标**: 与去年同期对比

**操作步骤**:
1. 使用时间轴滑块定位到去年同期
2. 记录去年该周的赔付率
3. 切换到今年同期
4. 对比赔付率变化

**决策依据**:
- 今年低于去年 → 风控改善
- 今年高于去年 → 分析恶化原因
- 波动规律相似 → 季节性因素

### 场景5: 目标监控
**目标**: 跟踪全年赔付率目标达成情况

**操作步骤**:
1. 查看经营摘要中的高风险周数统计
2. 观察图表中最新周的赔付率值
3. 对比年度目标（如68%）
4. 评估剩余周数的调整空间

**决策依据**:
- 高风险周数增多 → 加强风控
- 最新周赔付率高于目标 → 需要调整
- 连续多周处于预警区 → 采取紧急措施

## 常见问题

### Q1: 为什么有些周没有数据点？
**A**: 可能原因：
1. 该周无业务数据（如系统停机、节假日）
2. 数据尚未录入系统
3. 数据质量问题被过滤

### Q2: 趋势线与实际值差距很大，是否正常？
**A**: 正常现象，可能原因：
1. 短期波动大（如大额赔案）
2. 季节性因素（如冬季事故率高）
3. 政策变化（如费改）

趋势线代表长期方向，短期偏离不影响整体判断。

### Q3: 点击风险点没有反应？
**A**: 当前版本下钻功能尚未完全集成，点击会：
1. 在浏览器控制台输出数据（按F12查看）
2. 弹窗提示下钻入口
3. 未来将自动跳转到详情页

### Q4: 如何导出图表？
**A**: 当前版本暂不支持图表导出，未来将支持：
- PNG/SVG图片导出
- CSV数据导出
- PDF报告生成

### Q5: 能否自定义阈值线？
**A**: 当前版本阈值固定为70%，未来将支持：
- 自定义阈值设置
- 多条阈值线（如60%、70%、80%）
- 动态阈值（基于历史均值）

### Q6: 为什么经营摘要中的数据与图表不一致？
**A**: 可能原因：
1. 年度累计签单保费取的是最新周（如第42周）的累计值，不是图表所有周的总和
2. 风险周数是历史累计统计，反映整个观察期的风险分布
3. 筛选条件可能影响数据范围

## 进阶技巧

### 技巧1: 快速定位拐点
使用趋势线与实际值的偏离度识别拐点：
- 实际值突然偏离趋势线 → 可能是拐点
- 连续多周偏离同方向 → 趋势反转确认

### 技巧2: 组合分析
结合其他模块进行多维分析：
- 趋势图 + 结构分析 → 定位高风险车型
- 趋势图 + 机构分析 → 定位低效机构
- 趋势图 + 客户分群 → 识别高风险客群

### 技巧3: 预测未来
基于趋势线预测未来赔付率：
1. 观察趋势线斜率
2. 计算未来N周的预测值
3. 评估风险可能性
4. 提前制定应对措施

### 技巧4: 异常预警
建立自己的预警规则：
- 连续3周赔付率>70% → 红色预警
- 单周赔付率>80% → 橙色预警
- 赔付率环比增长>5pp → 黄色预警
