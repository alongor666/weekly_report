# 率值指标区间状态值配置

**文档版本**: 1.1  
**创建日期**: 2025-01-20  
**最后更新**: 2025-01-20  
**适用系统**: 车险数据分析平台 (Insuralytics)

---

## 📋 概述

本文档专门定义车险数据分析平台中**率值类指标**的区间划分、状态等级和颜色编码规则。率值指标是业务分析的核心，直接反映经营质量和风险控制水平。

**更新说明**: 根据要求，所有指标的等级统一采用“保费时间进度达成率”的等级术语：卓越、健康、预警、危险、高危。对于原有6级指标，已合并调整为5级统一体系。

## 🎯 五级评分体系

所有率值指标均采用统一的**五级评分体系**，将业务指标转换为0-100标准化评分：

| 评分等级 | 评分范围 | 视觉反馈 | 颜色代码 | 业务含义 |
|---------|---------|---------|---------|----------|
| **卓越** | 95-100 | 深绿色 | #2E7D32 | 表现卓越，超出预期 |
| **健康** | 86-94 | 浅绿色 | #4CAF50 | 表现良好，符合预期 |
| **预警** | 70-85 | 蓝色 | #1976D2 | 表现正常，可接受（需关注） |
| **危险** | 40-69 | 黄色/橙色 | #FBC02D / #F57C00 | 需要关注，存在风险 |
| **高危** | 0-39 | 红色 | #D32F2F | 严重风险，需立即处理 |

## 📊 核心率值指标区间设定

### 1. 满期边际贡献率 (contribution_margin_ratio)

**指标性质**: 正向指标（越高越好）  
**业务含义**: 反映业务盈利能力的核心指标  
**计算公式**: `100% - 变动成本率`

| 原始值范围 | 评分范围 | 等级 | 色彩 | 业务解读 |
|-----------|---------|------|------|----------|
| > 12% | 95-100 | 卓越 | 深绿 #2E7D32 | 盈利能力卓越 |
| 8-12% | 86-94 | 健康 | 浅绿 #4CAF50 | 盈利能力健康 |
| 6-8% | 70-85 | 预警 | 蓝色 #1976D2 | 盈利能力正常 |
| 4-6% | 40-69 | 危险 | 黄色 #FBC02D | 盈利空间有限 |
| < 4% | 0-39 | 高危 | 红色 #D32F2F | 亏损状态（合并0-4%和<0%） |

### 2. 保费时间进度达成率 (premium_time_progress_achievement_rate)

**指标性质**: 正向指标（越高越好）  
**业务含义**: 保费目标完成进度与时间进度的匹配度  
**计算公式**: `当期签单保费 / (年度目标保费 / 365 * 当期天数)`

| 原始值范围 | 评分范围 | 等级 | 色彩 | 业务解读 |
|-----------|---------|------|------|----------|
| ≥ 110% | 95-100 | 卓越 | 深绿 #2E7D32 | 超额完成目标 |
| 100-110% | 86-94 | 健康 | 浅绿 #4CAF50 | 按计划完成 |
| 90-100% | 70-85 | 预警 | 黄色 #FBC02D | 略低于预期 |
| 80-90% | 40-69 | 危险 | 橙色 #F57C00 | 明显滞后 |
| < 80% | 0-39 | 高危 | 红色 #D32F2F | 严重滞后 |

### 3. 满期赔付率 (loss_ratio)

**指标性质**: 逆向指标（越低越好）  
**业务含义**: 赔款支出占保费收入的比例  
**计算公式**: `满期赔付 / 满期保费`

| 原始值范围 | 评分范围 | 等级 | 色彩 | 业务解读 |
|-----------|---------|------|------|----------|
| < 50% | 95-100 | 卓越 | 深绿 #2E7D32 | 赔付控制优秀 |
| 50-60% | 86-94 | 健康 | 浅绿 #4CAF50 | 赔付控制良好 |
| 60-70% | 70-85 | 预警 | 蓝色 #1976D2 | 赔付控制正常 |
| 70-80% | 40-69 | 危险 | 黄色 #FBC02D | 赔付率偏高 |
| > 80% | 0-39 | 高危 | 红色 #D32F2F | 赔付率过高 |

### 4. 费用率 (expense_ratio)

**指标性质**: 逆向指标（越低越好）  
**业务含义**: 费用支出占保费收入的比例  
**计算公式**: `总费用 / 签单保费`

| 原始值范围 | 评分范围 | 等级 | 色彩 | 业务解读 |
|-----------|---------|------|------|----------|
| 0-7.5% | 95-100 | 卓越 | 深绿 #2E7D32 | 费用控制优秀 |
| 7.6-12.5% | 86-94 | 健康 | 浅绿 #4CAF50 | 费用控制良好 |
| 12.6-17.5% | 70-85 | 预警 | 蓝色 #1976D2 | 费用控制正常 |
| 17.6-22.5% | 40-69 | 危险 | 黄色 #FBC02D | 费用率偏高 |
| > 22.5% | 0-39 | 高危 | 红色 #D32F2F | 费用率过高 |

### 5. 变动成本率 (variable_cost_ratio)

**指标性质**: 逆向指标（越低越好）  
**业务含义**: 成本控制能力的综合体现  
**计算公式**: `满期赔付率 + 费用率`

| 原始值范围 | 评分范围 | 等级 | 色彩 | 业务解读 |
|-----------|---------|------|------|----------|
| < 60% | 95-100 | 卓越 | 深绿 #2E7D32 | 成本控制优秀 |
| 60-70% | 86-94 | 健康 | 浅绿 #4CAF50 | 成本控制良好 |
| 70-80% | 70-85 | 预警 | 蓝色 #1976D2 | 成本控制正常 |
| 80-90% | 40-69 | 危险 | 黄色 #FBC02D | 成本压力较大 |
| > 90% | 0-39 | 高危 | 红色 #D32F2F | 成本失控 |

### 6. 满期率 (maturity_ratio)

**指标性质**: 正向指标（越高越好）  
**业务含义**: 反映当期保费中已实现风险价值的部分  
**计算公式**: `满期保费 / 签单保费`

| 原始值范围 | 评分范围 | 等级 | 色彩 | 业务解读 |
|-----------|---------|------|------|----------|
| > 95% | 95-100 | 卓越 | 深绿 #2E7D32 | 风险实现度高 |
| 85-95% | 86-94 | 健康 | 浅绿 #4CAF50 | 风险实现度良好 |
| 70-85% | 70-85 | 预警 | 蓝色 #1976D2 | 风险实现度正常 |
| 50-70% | 40-69 | 危险 | 黄色 #FBC02D | 风险实现度偏低 |
| < 50% | 0-39 | 高危 | 红色 #D32F2F | 风险实现度不足 |

**指标性质**: 逆向指标（越低越好）  
**业务含义**: 衡量已满期保单的出险频率  
**计算公式**: `赔案件数 / 满期保单数`

## 🔧 技术实现

### 评分转换算法示例

```typescript
/**
 * 满期边际贡献率评分转换（统一5级）
 * @param value 原始百分比值
 * @returns 标准化评分 (0-100)
 */
function convertContributionMarginRatio(value: number): number {
  if (value > 12) return 95 + Math.min((value - 12) * 5 / 8, 5); // 卓越
  if (value >= 8) return 86 + (value - 8) * 9 / 4;  // 健康
  if (value >= 6) return 70 + (value - 6) * 16 / 2; // 预警
  if (value >= 4) return 40 + (value - 4) * 30 / 2; // 危险
  return Math.max(0, 40 - Math.abs(value - 4) * 40 / 8); // 高危
}

/**
 * 满期赔付率评分转换（逆向指标）
 * @param value 原始百分比值
 * @returns 标准化评分 (0-100)
 */
function convertLossRatio(value: number): number {
  if (value < 50) return 95 + Math.min((50 - value) * 5 / 10, 5); // 卓越
  if (value <= 60) return 86 + (60 - value) * 9 / 10;  // 健康
  if (value <= 70) return 70 + (70 - value) * 16 / 10; // 预警
  if (value <= 80) return 40 + (80 - value) * 30 / 10; // 危险
  return Math.max(0, 40 - (value - 80) * 40 / 20);     // 高危
}
```

### 综合评分计算

**多维雷达图综合评分** = 5个核心率值维度评分的算术平均值

```typescript
const overallScore = Math.round(
  (contributionMarginScore + progressScore + lossRatioScore + 
   claimRatioScore + expenseRatioScore) / 5
);
```

**注意事项**:
1. 率值指标是业务分析的核心，直接影响经营决策
2. 所有阈值基于车险行业经验和历史数据制定
3. 逆向指标（如赔付率、费用率）采用反向评分逻辑
4. 颜色编码遵循国际通用的红绿灯预警体系
5. 评分算法采用线性插值，确保平滑过渡
6. 统一等级后，所有指标均为5级体系，便于跨指标比较
