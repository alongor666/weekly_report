## 附录：分析说明

### 数据来源
- 数据源：2025年保单变动成本明细表
- 分析周期：第28周 - 第43周
- 筛选条件：is_new_energy_vehicle = True 且 business_type_category 包含"货车"
- 统计口径：累计值（非当周发生值）

### 关键指标定义
- **赔付率** = 已报告赔款 / 满期保费 × 100%
- **费用率** = 费用总额 / 签单保费 × 100%
- **边际贡献率** = 100% - 赔付率 - 费用率
- **出险率** = 赔案件数 / 保单件数 × 100%
- **案均赔款** = 已报告赔款 / 赔案件数
- **单均保费** = 签单保费 / 保单件数

### 风险等级标准
- 🟢 良好：赔付率 < 60%
- 🟡 关注：60% ≤ 赔付率 < 70%
- 🟠 预警：70% ≤ 赔付率 < 80%
- 🔴 高危：赔付率 ≥ 80%
